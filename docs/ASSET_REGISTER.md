## Notes

- The current rehaul removes visible story captions and improves motion/script behavior first.
- The prior generated character and vehicle PNGs remain in the repository as legacy assets until the selected Kenney packs are imported and mapped to exact files.
- Do not add random GitHub images or unclear-license files to this game.
