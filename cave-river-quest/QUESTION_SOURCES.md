# Cave River Quest Question Sources

The in-game questions are original, but the topic mix is based on public Grade 4 curriculum guidance:

- Common Core Grade 4 Mathematics: operations, factors/multiples, fractions, measurement/time, and geometry.
  - https://www.thecorestandards.org/Math/Content/4/introduction/
- Common Core Grade 4 English Language Arts: inference, context clues, vocabulary, conventions, and sentence revision.
  - https://www.thecorestandards.org/ELA-Literacy/

Question design target: Grade 4-friendly, multi-step where practical, readable in a short game dialog, and varied across maths, logic, reading, vocabulary, grammar, geometry, and judgment.
